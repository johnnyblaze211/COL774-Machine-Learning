For running code using python:

For Q1 (Decision Trees), run:
python3 dt.py <path_to_train_file> <path_to_test_file> <path_to_validation_file> <part>

For Q2 (Neural Networks) run:
python3 nn.py <tpath_to_train_file> <path_to_test_file> <part>

{Note: part must be in a-d for Q1 and in a-f for Q2}


For running using bash:

Question 1
./run.sh 1 <path_of_train_data> <path_of_test_data>  <path_of_validation_data>  <part_num>
Here, 'part_num' can be a-d.

Question 2
./run.sh 2 <path_of_train_data> <path_of_test_data> <part_num>

Here, 'part_num' is part number which can be a-g.
